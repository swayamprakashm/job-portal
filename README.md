
# Job Portal – React Application

A modern **Job Portal web application** built with **React** that allows users to browse job listings, view detailed job descriptions, apply for jobs, and save jobs for later.

The project focuses on building a clean UI with **Claymorphism design**, React routing, and local storage functionality.

---

## Features

• Browse available job listings
• View detailed job description
• Save jobs for later
• Apply for jobs with popup notification
• Saved jobs page with job logos
• Responsive job cards layout
• Claymorphism UI design
• React Router navigation

## Pages

### Landing Page

Hero section with introduction and navigation to browse jobs.

### Jobs Page

Displays available jobs in a grid layout with search functionality.

### Job Details Page

Shows complete information about a selected job including:

* Company logo
* Job title
* Company name
* Location
* Job type
* Description
* Apply button
* Save job option

### Saved Jobs Page

Displays jobs saved by the user with:

* Company logo
* Job title
* Company name
* View details button

## Installation

Clone the repository

```
git clone https://github.com/yourusername/job-portal.git
```

Navigate to project folder

```
cd job-portal
```

Install dependencies

```
npm install
```

Start development server

```
npm run dev
```
